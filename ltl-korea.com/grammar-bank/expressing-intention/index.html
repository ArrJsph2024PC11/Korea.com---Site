    INTENTION IN KOREAN

    Do you plan to do something this weekend? Are you attempting to achieve new goals?

    We will teach you all sentence structures to express your intentions and plans in Korean.

    INTRODUCTION

    There are several ways to express plan or intension in Korean.

    The sentence structure usually uses ‘하려고 있다’ or ‘~ 하는 중 이다’ which means ‘in the middle of attempting’.

    This structure is used with several different verbs, including:

    • 하다 (to do)
    • 생각하다 (to think)
    • 고려하다 (to consider)
    • 계획하다 (to plan)
    • 의도하다 (to intend)

    As usual, for the object being intended/planned, the article ‘를’ is used if the noun ends with a vowel, and an article ‘을’ is used if the noun ends with a consonant. 

    WITH 하다 (to do)

    Expressing intention in Korean with the verb 하다 (to do) is done by following one of the structures:

    PATTERN

    Subject + Object + 를/을 하려고 하다

    Subject + Object + 를/을 하려는 중 이다

    Let’s take the sentence “I am attempting to start exercising from today” as an example:

    WITH 생각하다 (to think)

    Expressing intention in Korean with the verb 생각하다 (to think) is done by following one of the structures:

    PATTERN

    Subject + Object + 를/을 (하려고) 생각하고 있다

    Subject + Object + 를/을 생각하고 있는 중 이다

    Let’s take the sentence “She is thinking about starting her trip tomorrow” as an example:

    WITH 고려하다 (to consider)

    Expressing intention in Korean with the verb 고려하다 (to consider) is done by following one of the structures:

    PATTERN

    Subject + Object + 를/을 (하려고) 고려하고 있다

    Subject + Object + 를/을 고려하고 있는 중 이다

    Let’s take the sentence “The teacher is considering to end the class soon” as an example:

    WITH 계획하다 (to plan)

    Expressing intention in Korean with the verb 계획하다 (to plan) is done by following one of the structures:

    PATTERN

    Subject + Object + 를/을 (하려고) 계획하고 있다

    Subject + Object + 를/을 계획하고 있는 중 이다

    Let’s take the sentence “Young Hee is planning on attending Cheol Su’s birthday party” as an example:

    WITH 의도하다 (to intend)

    Expressing intention in Korean with the verb 의도하다 (to intend) is done by following one of the structures:

    PATTERN

    Subject + Object + 를/을 (하려고) 의도하고 있다

    Subject + Object + 를/을 의도하고 있는 중 이다

    Let’s take the sentence “He is intending to propose to her” as an example:

    USING ~부터

    When expressing plan/intention in Korean, the start of the plan will often be indicated in the sentence by the use of ‘~부터 (bu teo)’.

    If the plan is about to start, words such as ‘곧 (got), 지금 (jigeum), 당장 (dang jang), 바로 (baro)’ adverbs are also used.

    FAQs

    How to say “plan to” do something in Korean?

    Expressing intention in Korean with the verb 계획하다 (to plan) is done by following one of the structures:

    Subject + Object + 를/을 (하려고) 계획하고 있다

    Subject + Object + 를/을 계획하고 있는 중 이다

    Example:

    영희는 철수의 생일파티에 참석하려고 계획하고 있다.

    Young Hee is planning on attending Cheol Su’s birthday party

    How to say “consider to” do something?

    Expressing intention in Korean with the verb 고려하다 (to consider) is done by following one of the structures:

    Subject + Object + 를/을 (하려고) 고려하고 있다

    Subject + Object + 를/을 고려하고 있는 중 이다. 

    Example:

    선생님은 곧 수업을 끝내려고 고려하고 있다. 

    The teacher is considering to end the class soon

    How to say “think to” do something?

    Expressing intention in Korean with the verb 생각하다 (to think) is done by following one of the structures:

    Subject + Object + 를/을 (하려고) 생각하고 있다

    Subject + Object + 를/을 생각하고 있는 중 이다.. 

    Example:

    그녀는 내일부터 여행을 시작하려고 생각하고 있다.

    She is thinking about starting her trip tomorrow

    How to use 부터?

    When expressing plan/intention in Korean, the start of the plan will often be indicated in the sentence by the use of ‘~부터 (bu teo)’.

    Example:

    그녀는 다음 주 부터 새로운 학기를 시작하려고 계획 중 이다. 

    She is planning to start a new semester from next week.
